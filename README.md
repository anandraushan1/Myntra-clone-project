# 🛍️ Myntra Clone (HTML, CSS, JavaScript)

A front-end clone of the popular e-commerce platform **Myntra**, built using **HTML, CSS, and JavaScript**. This project replicates the look and feel of an online fashion shopping website with interactive features and a responsive design.

---

## 🚀 Features

- 🏠 Home page with banners and product categories  
- 🛒 Add to Cart functionality  
- 🔍 Product listing with search and filtering  
- 📄 Product details page  
- ❤️ Wishlist feature (optional)  
- 📱 Fully responsive (Mobile, Tablet, Desktop)  
- 🎨 Modern UI inspired by Myntra  

---

## 🛠️ Tech Stack

- **HTML5** – Page structure  
- **CSS3** – Styling and layout (Flexbox & Grid)  
- **JavaScript (Vanilla JS)** – Interactivity and logic
