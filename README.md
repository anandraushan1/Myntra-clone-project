# Myntra-clone-project


🛍️ Myntra Clone — Frontend E-commerce Project

A responsive and interactive Myntra-inspired e-commerce web application built using HTML, CSS, and JavaScript.
This project demonstrates strong frontend development skills, UI/UX design understanding, and dynamic functionality using vanilla JavaScript.

🚀 Features
🏠 Homepage with banners and product sections
🔍 Search bar UI (basic functionality)
🛒 Add to cart feature (using JavaScript)
📦 Product listing with images, price, and details
❤️ Wishlist icon UI
📱 Fully responsive design (mobile + desktop)
🎨 Clean and modern UI similar to Myntra

🛠️ Tech Stack
HTML5 – Structure of the website
CSS3 – Styling and layout
JavaScript (Vanilla JS) – Functionality and interactivity
